Enumerator
GAME_START 

game just started

GAME_SERVE 

serving cards to players and putting cards onto table

GAME_CYCLE 

cycle through player turns, check if a player do his move, if must serve cards or game's ended

GAME_WAITING 

waiting for player move

GAME_END 

game ended append table cards to last taker and check who's the winner
